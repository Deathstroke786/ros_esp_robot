# ESP32 BASED ROS MOBLIE ROBOT

Basically I'm here making ros based mobile robot which works on esp32 microcontroller.

Project

  TurtleSim Obstical Avoiding using real World Sensor

Hardware Required

    ESP32
    Robotic Car Kit
    Battery and Charger
    Ultrasonic Sensor
    Motor Driver ( L298D)
    Jumper Wires

Software Based

    Ubuntu 20.04 (LTS)
    ROS1 - Noetic
    Python 3.6
    Arduino Ide
    Vscode
